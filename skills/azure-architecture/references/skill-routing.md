# Azure Skill Routing

Use for `/azure route`.

## Route To Specialized Azure Skills

- `azure-prepare`: create or modernize app deployment assets, Bicep/Terraform,
  Dockerfiles, `azure.yaml`, or hosting plan.
- `azure-deploy`: execute an already prepared deployment.
- `azure-validate`: run pre-deployment readiness checks.
- `azure-diagnostics`: troubleshoot live production issues.
- `azure-reliability`: assess and improve PaaS reliability posture.
- `azure-cost`: query or optimize Azure spending.
- `azure-rbac`: choose or assign least-privilege Azure RBAC roles.
- `azure-compliance`: run compliance and security audits.
- `azure-resource-lookup`: inventory Azure resources.
- `azure-resource-visualizer`: diagram existing Azure resources.
- `azure-kubernetes`: plan or create AKS clusters.
- `azure-ai`, `azure-aigateway`, `microsoft-foundry`: Azure AI, AI gateway,
  Foundry, model, or agent work.
- `entra-app-registration` or `entra-agent-id`: Entra app, OAuth, or agent
  identity setup.

## Stay In This Skill

Stay in `azure-architecture` when the question is architectural: target state,
landing zone, governance, Azure WAF/CAF alignment, service choice rationale,
migration positioning, or routing.
