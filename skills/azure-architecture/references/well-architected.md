# Azure Well-Architected Review

Use for `/azure waf`.

## Pillars

- Reliability: failure modes, availability zones, region strategy, recovery
  objectives, backups, and operational validation
- Security: identity, access, network isolation, data protection, monitoring,
  threat detection, and secure operations
- Cost Optimization: right sizing, scaling, reservations, budgets, tagging,
  ownership, and waste reduction
- Operational Excellence: IaC, deployment, observability, incident response,
  runbooks, and change management
- Performance Efficiency: capacity, scaling, latency, caching, data access, and
  service limits

## Output Expectations

- State pillar findings with severity and evidence.
- Separate architecture recommendations from execution tasks.
- Route cost queries to `azure-cost`, reliability remediation to
  `azure-reliability`, validation to `azure-validate`, and deployment to
  `azure-deploy`.
